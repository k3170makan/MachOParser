# MachOParser
Quick Dirty python API for pulling info out of MachO files
# Usage
```
./minmal_parser [MachO Binary]

[*] Magic Number:['ca', 'fe', 'ba', 'be']
[*] CPU Sub Type:['0', '0', '0', 'c']
[*] Number of Load Commands:(16384,)
[*] File Type:['0', '0', '0', '9']
[*] CPU Type:['0', '0', '0', '2']
[*] Flags:['0', '0', '0', 'e']
[*] Size of Load Command Field:(1502560,)
```
